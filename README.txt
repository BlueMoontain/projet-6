Poch'Lib - Book Management App


Poch'Lib is a web application for managing books, developed for "La Plume enchant√©e," a local bookstore in Nice.

Installation
To run this project locally, follow these steps:

Clone the repository: git clone https://github.com/BlueMoontain/projet-6
Navigate to the project directory: cd poch-lib
Install dependencies: npm install
Start the application: npm start
Make sure you have Node.js and npm installed on your machine.

Usage
Click the "Ajouter un livre" button to search for books online.
Fill in the "Titre du livre" and "Auteur" fields in the search form.
Click "Rechercher" to retrieve books from the Google Books API.
Click on Bookmark icon to add a book to your reading list (Poch'Liste).
You can view and manage your reading list below the search results.
Click the Poubelle icon to remove a book from your reading list.
